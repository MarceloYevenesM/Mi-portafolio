Pokédex - React

Proyecto realizado en React, conectando a PokéApi para obtener datos de la primera generación de Pokemons, incluye buscador, paginación, detalles del Pokémon. Puede ser visualizado desde cualquier dispositivo con acceso a la web.

TalentWeek Coderhouse - React

Proyecto realizado en React para la feria laboral 2022 de Coderhouse como proyecto freelance, fue realizado en conjunto a un equipo de estudiantes del curso de React y de diseño UX/UI de Coderhouse. Puede ser visualizado desde cualquier dispositivo con acceso a la web.

FitnessCoffee - React

Proyecto realizado en HTML y CSS, como proyecto final de Nucba. El proyecto consiste en una página de presentación de una Cafeteria de productos saludables y visible dede cualquir dispositivo con acceso a la web.
